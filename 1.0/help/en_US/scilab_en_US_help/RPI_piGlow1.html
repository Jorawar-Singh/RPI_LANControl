RPI_piGlow1

Function to modify a single LED on the piGlow

Calling Sequence

RPI_piGlow1(leg,ring,intensity)

Parameters

leg :

leg number

ring :

ring number

intensity:

brightness from 0 (off) to 255 (full)

Description

The function modifies the brightness of the single specified LED.

Examples

RPI_piGlow1(0,0,125)
