RPI_I2CRead

Function to read device

Calling Sequence

RPI_I2CRead(fd)

Parameters

fd:

file descriptor obtained from RPI_I2CSetup

Description

Function for a simple device read. Some devices present data when you read them without having to do any register transactions.

Examples

RPI_I2CRead(13)
