RPI_pwmRange

Function to set the divisor for the PWM clock.

Calling Sequence

RPI_pwmRange(value)

Parameters

value:

maximum of the range for the PWM generator.

Description

This function sets the range register in the PWM generator. The default is 1024.

Examples

RPI_pwmRange(512)
