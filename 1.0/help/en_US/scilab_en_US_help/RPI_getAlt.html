RPI_getAlt

Function to get the current mode of pin

Calling Sequence

s=RPI_getAlt(pin)

Parameters

pin:

The pin number to read the voltage from. Numbering sequence to be followed as initiated using RPI_pinNumbering function.

Description

This function returns the current mode of the pin. Output will be 0 for INPUT, 1 for OUTPUT, 2 for PWM_OUT and 3 for CLOCK.

Examples

s=RPI_getAlt(0)
