RPI_lcdSendCommand

Function to send command to lcd

Calling Sequence

RPI_lcdSendCommand(fd,command)

Parameters

fd :

file-descriptor obtained using RPI_lcdInit function

command:

A command, binary, to submit to the LCD.

Description

This function sends any arbitrary command to the LCD.

Examples

RPI_lcdSendCommand(13,00)
