RPI_gertboardSPISetup

Function to initialize SPI bus communication with gertboard

Calling Sequence

RPI_gertboardSPIsetup()

Description

This function must be called to initialize the SPI bus to communicate with the Gertboards ADC and DAC chips.

Examples

RPI_gertboardSPISetup()
