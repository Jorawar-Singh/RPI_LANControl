RPI_digitalWrite

Function to set digital voltage at the given pin

Calling Sequence

RPI_digitalWrite(pin,Volt)

Parameters

pin :

The pin number to set the voltage at. Numbering sequence to be followed as initiated using RPI_pinNumbering function.

Volt:

Digital voltage to set. 0 (for logic-LOW) or 1 (for logic-HIGH)

Description

This funtion sets the voltage at the given pin. 0v for logic 0 and 3.3v for logic 1. Note: pin should be set for output mode using RPI_pinMode.

Examples

RPI_digitalWrite(0,1)
