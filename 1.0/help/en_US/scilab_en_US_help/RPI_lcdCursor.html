RPI_lcdCursor

Function to turn cursor on/off

Calling Sequence

RPI_lcdCursor(fd,state)

Parameters

fd :

file-descriptor obtained using RPI_lcdInit function

state:

0 (for off) or 1 (for on)

Description

This function sets the visibility of cursor.

Examples

RPI_lcdCursor(13,1)
