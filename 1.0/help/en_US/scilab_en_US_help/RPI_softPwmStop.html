RPI_softPwmStop

Function to stop the software pwm on the pin

Calling Sequence

RPI_softPwmStop(pin)

Parameters

pin:

pin to stop. Numbering sequence to follow as initiated using RPI_pinNumbering

Description

This function stops the software pwm on the specified pin.

Examples

RPI_softPwmStop(1)
