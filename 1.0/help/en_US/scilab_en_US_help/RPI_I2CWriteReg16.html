RPI_I2CWriteReg16

Function to write 16-bit data to register

Calling Sequence

RPI_I2CWriteReg16(fd,reg,data)

Parameters

fd :

file descriptor obtained from RPI_I2CSetup

reg :

integer register to write data at

data:

integer data to write to device

Description

This function writes a 16-bit data value into the device register indicated.

Examples

RPI_I2CWriteReg16(13,0x40,100)
