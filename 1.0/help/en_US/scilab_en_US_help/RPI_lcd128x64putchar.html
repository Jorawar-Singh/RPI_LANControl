RPI_lcd128x64putchar

Function to print a single character.

Calling Sequence

RPI_lcd128x64putchar(x,y,c,bg,fg)

Parameters

x :

integer, x coordinate of starting point

y :

integer, y coordinate of starting point

c :

integer, from 0 to 255, character to print

bg:

integer, background color

fg:

integer, foreground color

Description

This function prints a single character to the screen.

Examples

RPI_lcd128x64putchar(0,1,128,1,0)
